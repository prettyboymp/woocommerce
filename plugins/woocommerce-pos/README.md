# WooCommerce POS

A Point of Sale extension for WooCommerce.

## Description

WooCommerce POS extends WooCommerce with Point of Sale functionality, allowing you to process in-store sales and send customized email receipts to customers.

## Features

- Process in-store sales through WooCommerce
- Send customized email receipts for POS orders
- Seamless integration with WooCommerce core

## Requirements

- WordPress 6.0+
- WooCommerce 8.0+
- PHP 7.4+

## Installation

1. Upload the `woocommerce-pos` zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings through WooCommerce > Settings > Emails
